## paas-go-example

## Built With

* Go 1.9.4 or higher.

## How to run the app

* Clone the repository.
* Set an environmental variable called **PORT**. (export PORT=1234)
* Change directory into the repo directory.
* Execute `go build`
* You can access the app via your browser `http://localhost:1234`
